Detailed Description

template<typename CoordinateT = Coordinate<DistanceM>, typename AllocatorT = AllocatorPSRAM<CoordinateT>>
class tinyrobotics::Path< CoordinateT, AllocatorT >

A simple path class that holds a sequence of waypoints. Each waypoint is e.g. a coordinate (e.g., x,y,z) that the robot should follow. The path can be used for navigation, motion planning, or trajectory execution. It provides methods to add waypoints.

Template Parameters
TCoordinate e.g. GPSCoordinate, Coordinate, ScheduledWayPoint, WayPointAndSpeed etc.

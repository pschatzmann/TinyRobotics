14/**
15 * @class RangeSensor
16 * @ingroup sensors
17 * @brief Generic range sensor abstraction for LIDAR, ultrasonic, or similar
18 * sensors.
19 *
20 * This template class models a simple range sensor, such as a LIDAR or
21 * ultrasonic sensor, for robotics applications. It tracks the measured distance
22 * to an obstacle, the angle (bearing) of the obstacle relative to the sensor,
23 * and the transform from the sensor's local frame to the world frame. The class
24 * provides methods to set and retrieve the measured distance and bearing,
25 * update the sensor-to-world transform, and compute the world-frame coordinates
26 * of detected obstacles.
27 *
28 * ## Features
29 * - Supports any distance unit via template parameter (default: meters)
30 * - Publishes distance, angle, and obstacle position as messages (for
31 * integration with messaging framework)
32 * - Computes obstacle coordinates in the world frame using a 2D transform
33 * - Validity checks for sensor readings (distance > 0)
34 * - Can be used for navigation, mapping, and obstacle avoidance
35 *
36 * ## Usage Example
37 * @code
38 * Transform2D tf = ...; // Sensor-to-world transform
39 * RangeSensor<> sensor(tf, 0.0f); // 0 degrees = forward
40 * sensor.begin();
41 * sensor.setObstacleDistance(1.5f); // Set measured distance
42 * if (sensor) {
43 * Coordinate<DistanceM> obs;
44 * if (sensor.getObstacleCoordinate(obs)) {
45 * // Use obs for navigation
46 * }
47 * }
48 * @endcode
49 *
50 * ## Template Parameters
51 * @tparam T Distance type (default: DistanceM)
52 *
53 * ## Methods
54 * - setObstacleDirectionDegree(deg): Set obstacle bearing in degrees
55 * - setObstacleDistance(distance): Set measured distance
56 * - setObstacle(degree, distance): Set both bearing and distance
57 * - setTransform(tf): Set sensor-to-world transform
58 * - getObstacleCoordinate(result): Compute world-frame obstacle coordinate
59 * - hasObstacle(): True if a valid obstacle is detected
60 *
61 * ## Applications
62 * - LIDAR, ultrasonic, or IR range sensing
63 * - Robot navigation and mapping
64 * - Obstacle detection and avoidance
65 *
66 * @author Phil Schatzmann
67
68 */

129 // If an obstacle is detected (distance > alertDistance), call the callback
130 if (callback_ && distanceM > alertDistanceM_ &&

136 Coordinate<T> obstacle;
137 if (getObstacleCoordinate(obstacle)) {
138 // publish distance

141 sendMessage(msgDistance);

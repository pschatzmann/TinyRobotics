TinyRobotics

TinyRobotics is a modular, header-only C++ library for Arduino and embedded systems, providing essential building blocks for robotics, navigation, mapping, and sensor integration. The library is designed for flexibility and extensibility, supporting a variety of coordinate systems, path planning algorithms, and sensor types.

<a href="src/TinyRobotics" >Features</a>

The following base functionality is automatically available:

  • **Units**: Strongly-typed units for distance, time, speed, and angle to ensure type safety and clarity.
  • **Coordinate Systems**: 2D/3D coordinates, GPS coordinates, NMEA sentence parsing, local cartesian frames, and hierarchical frame management (SE(2)/SE(3)).
  • **Mapping**: Grid maps (occupancy/value), path maps (graph-based), and 3D point clouds for environment modeling and SLAM.
  • **Path Planning**: Generic A* and Dijkstra algorithm, path representation, and planning utilities for navigation and obstacle avoidance.
  • **Control**: Core control algorithms including PID controllers, Kalman filters, moving averages, schedulers, and the MotionController for path following and vehicle control.
  • **Sensors**: RangeSensor abstraction, Camera scenarios and WheelEncoder
  • **Odometry**: 2D and 3D odometry modules for incremental position, orientation, and distance estimation. Provides robust pose tracking for mobile robots and vehicles.
  • **IMU**: 2D and 3D Inertial Measurement Unit (IMU) abstraction for embedded robotics. Provides real-time access to orientation, acceleration, and angular velocity. Integrates seamlessly with control, navigation, and SLAM modules for feedback and state estimation.
  • **Fusion**: Sensor fusion algorithms for combining IMU, odometry, and other sensor data to estimate robot pose, velocity, and orientation in 2D and 3D.
  • **SLAM**: Simultaneous Localization and Mapping (SLAM) for real-time robot pose estimation and map building. Includes frame transforms, exploration, and modular integration with sensors and vehicles.
  • **Motors**: Drivers and abstractions for stepper, DC, and RC servo motors. .
  • **Vehicles**: High-level abstractions for controlling cars (4WD, Ackerman), quadrotors, airplanes, and boats.
  • **Utilities**: Logger, Streams, Buffers, serialization and memory allocators for embedded systems.

The following extended functionality can be included optionally:

  • **Communication**: Message-based framework supporting UDP, ESPNow, Serial, LoRa, IEEE802_15_4, and more for telemetry, remote control, and inter-robot messaging.
  • **Concurrency**: Thread-safe queues, buffers, mutexes, and RTOS integration for multitasking and synchronization.

Installation

For Arduino, you can download the library as zip and call include Library -> zip library. Or you can git clone this project into the Arduino libraries folder e.g. with

cd ~/Documents/Arduino/libraries
git clone https://github.com/pschatzmann/TinyRobotics.git

Dependencies

You also need to install the follwing Arduino libraries:

  • Servo / ESP32Servo
  • FastAccelStepper

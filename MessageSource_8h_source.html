17 * MessageSource manages a list of MessageHandler objects and provides methods
18 * to add, clear, and forward messages to all registered handlers. This enables
19 * a flexible publish/subscribe or observer pattern for distributing messages
20 * (such as sensor data, commands, or events) to multiple consumers in a
21 * robotics system.
22 *
23 * Typical usage:
24 * @code
25 * MessageSource source;
26 * source.subscribe(handler1);
27 * source.subscribe(handler2);
28 * source.sendMessage(msg); // Forwards to all handlers
29 * @endcode
30 *
31 * Extend this class to implement custom message-producing components (e.g.,
32 * sensors, controllers) that need to broadcast messages to other system parts.
33 */

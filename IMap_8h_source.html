5/**
6 * @class IMap
7 * @ingroup maps
8 * @brief Abstract interface for 2D grid maps in TinyRobotics.
9 *
10 * Provides a generic interface for occupancy grid maps, supporting cell access,
11 * coordinate transforms, and neighbor queries. All map implementations (e.g., GridBitMap)
12 * must inherit from this interface and implement all pure virtual methods.
13 *
14 * @tparam T Scalar type for coordinates and math (default: DistanceM)
15 *
16 * ### Required Methods
17 * - getXCount(), getYCount(): Map dimensions in cells
18 * - getResolution(): Cell size in meters
19 * - getNeighbors(): Neighboring cell coordinates for pathfinding
20 * - isValid(): Check if a coordinate is inside the map
21 * - getCell(): Get cell state (FREE, OCCUPIED, UNKNOWN) by index
22 * - toWorld(): Convert cell indices to world coordinates
23 */

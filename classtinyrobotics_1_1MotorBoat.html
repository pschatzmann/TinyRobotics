Detailed Description

template<typename MotorMT = BrushedMotor, typename ServoMT = ServoMotor>
class tinyrobotics::MotorBoat< MotorMT, ServoMT >

Motor boat with 1 drive motor and 1 rudder servo.

This class abstracts a simple motor boat:

  • 1 drive motor (HBridge)
  • 1 rudder servo (ServoMotor)

Usage Example:

MotorBoat boat;
boat.setPins(5, 6, 9, 10); // in1, in2, pwm, rudderPin
boat.setThrottle(70); // 70% throttle
boat.setRudder(20); // 20 degrees left
boat.stop(); // brake
